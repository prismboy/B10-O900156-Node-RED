近畿大学　情報メディアプロジェクトⅡ
====================================

### デモンストレーション環境

近大　情報メディアプロジェクトⅡにてデモンストレーションを実演するための環境

### コンテンツのご紹介

1. Node-REDフロー
 - 各演習で作成するフローの見本
1. 各ハンズオンごとのサンプルコード集
 - 各ハンズオンごとにサンプルコード類をZIPファイルとして提供。/materials/index.html にてリンクを公開
